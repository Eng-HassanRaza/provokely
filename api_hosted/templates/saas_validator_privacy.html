SaaSProof

Privacy Policy for SaaS Validator Extension

Last Updated: January 2025

Overview

SaaS Validator is a market research tool that helps entrepreneurs validate SaaS ideas by analyzing public Reddit discussions and market trends.

How the Extension Works

SaaS Validator offers two modes:

1. Hosted Service (Default - Recommended)

  • You sign in with your email
  • AI analysis is powered by our servers
  • No API keys needed

2. Own API Keys (Advanced)

  • You provide your own OpenAI API key
  • AI analysis happens directly between your browser and OpenAI
  • Full control over API usage

Data Collection

Hosted Service Mode

What we collect:

  • Email address (for authentication)
  • Project names and validation data (stored on our servers)
  • Public Reddit posts and comments (scraped from Reddit.com)
  • Google search results (scraped from Google.com)
  • Usage statistics (for service improvement)

What we DON'T collect:

  • Personal browsing history
  • Data from websites other than Reddit and Google search
  • Credit card information (no payment required currently)
  • Personal messages or private data

Own API Keys Mode

What we collect:

  • Project data (stored locally in your browser only)
  • No server-side data collection

What we DON'T collect:

  • Your API keys are NEVER sent to our servers
  • No usage tracking
  • No personal information

Data Storage

Hosted Service

  • Email and authentication tokens stored securely on our servers
  • Project data stored on our servers (encrypted at rest)
  • Local browser storage used for temporary data
  • Data retained until you delete your account

Own Keys Mode

  • All data stored locally in your browser only
  • Nothing sent to our servers
  • Your API keys stored encrypted in Chrome's local storage

Third-Party Services

OpenAI API

  • Used to analyze market data and generate insights
  • Data sent: Reddit discussions, market research data, prompts
  • OpenAI's privacy policy: https://openai.com/policies/privacy-policy

Reddit & Google

  • We extract publicly available information only
  • No authentication required
  • No personal data accessed

Permissions Justification

  • storage: Store your settings and project data locally
  • tabs: Open Reddit threads and Google searches for automated research
  • activeTab: Access current tab only when you click the extension
  • sidePanel: Display the extension interface
  • host_permissions:
    • google.com: Extract public search results
    • reddit.com: Extract public discussions
    • provokely.com: Connect to our hosted service (for authentication and AI processing)

Data Sharing

We do NOT share, sell, or rent your data to third parties.

Data is only sent to:

  • OpenAI API (for AI analysis of market research data)
  • Our backend servers (provokely.com) - only in Hosted Service mode

User Rights

You have the right to:

  • Access your data at any time
  • Delete all stored data via extension settings
  • Export your validation reports as JSON
  • Switch between Hosted Service and Own Keys modes
  • Revoke the extension's permissions at any time
  • Delete your account and all associated data

To delete your data:

  • Hosted Service: Contact us at support@provokely.com
  • Own Keys: Uninstall the extension

Security

  • Authentication tokens use industry-standard JWT
  • All API communications use HTTPS encryption
  • Passwords are never stored (email-only authentication)
  • Local data encrypted using Chrome's secure storage

Children's Privacy

This extension is not intended for users under 13 years of age. We do not knowingly collect data from children.

Changes to This Policy

We will notify users of any significant changes via:

  • Extension update notes
  • Email notification (for Hosted Service users)
  • Notice in the extension interface
