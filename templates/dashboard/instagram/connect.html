Development Setup Required

To use Instagram OAuth, you need to:

  1. Create an Instagram app at Facebook Developers
  2. Add "Instagram Basic Display" product
  3. Update credentials in config/settings.py
  4. Configure redirect URI: http://localhost:8000/dashboard/instagram/callback/

See INSTAGRAM_SETUP.md for detailed instructions.
